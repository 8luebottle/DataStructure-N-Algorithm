# Sum up all of the numbers
Sum will sum up all of the numbers passed in and return the result.

**Example 1**
```go
Sum([]int{1, 2, 3, 4, 5}) // 15
```

**Example 2**
```go
Sum([]int{3, 3, 3, 3, 3}) // 15
```

**Example 3**
```go
Sum([]int{3, 5, 3, 5, 3}) // 19
```

**Example 4**
```go
Sum([]int{4, 2, 22, -10, 8}) // 26
```

**Example 5**
```go
Sum(nil) // 0
```

**Example 6**
```go
Sum(int[]{}) // 0
```
